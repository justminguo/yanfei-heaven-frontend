魔法娃娃

對照 381 的 etcitem_doll_type / etcitem_doll_power，整理娃娃效果與用途

🎎 系統說明

天堂1的魔法娃娃不只是外觀收藏，而是會直接影響角色實戰體感的重要系統。381 目前可確認至少有兩張核心表：etcitem_doll_type 與 etcitem_doll_power。前者偏向娃娃本體與需求，後者偏向效果模板。

💡 之後網站會把娃娃本體與效果模板對起來，讓玩家能直接查「這隻娃娃到底有什麼功能」。

📊 實際資料表欄位

etcitem_doll_type

娃娃本體資料

  • itemid：對應道具 ID
  • nameid：顯示名稱字串
  • powers：效果關聯代碼
  • note：效果備註說明
  • need / count：需求道具與數量
  • time：持續時間
  • gfxid：外觀圖像
  • mode：模式/分類代碼
etcitem_doll_power

娃娃效果模板

  • id：效果模板 ID
  • classname：效果類型名稱
  • type1 / type2 / type3：效果數值或類型代碼
  • note：人類可讀的效果說明

🎀 已確認效果樣本

以下內容直接來自 381 的真資料，可先用來做玩家說明與後續查詢頁。

來源表樣本說明
etcitem_doll_powerDoll_Ac可對應防禦加成，已確認有 AC -1 / -2 / -3 / -4 等模板。
etcitem_doll_powerDoll_Hp可對應最大 HP 加成，已確認至少有 HP +10 樣本。
etcitem_doll_typeitemid / nameid / powers可用來把「娃娃本體」對到「實際效果模板」。
etcitem_doll_typetime可顯示娃娃持續時間，例：1800 秒。
etcitem_doll_typenote部分紀錄已直接寫出效果摘要，適合先做網站顯示。

📌 玩家怎麼使用這些資料

  • 先看娃娃本體：從 etcitem_doll_type 看名稱、需求與持續時間。
  • 再看效果模板：從 etcitem_doll_power 看真正加成，例如 AC、HP、MP、傷害或特殊功能。
  • 後續網站會做對照：把本體與效果模板合併，讓你直接查到「這隻娃娃到底加了什麼」。
