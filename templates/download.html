系統需求

最低需求
  • 作業系統Windows 7 / 10 / 11（64 位元）
  • 處理器Intel Core i3 或同等級
  • 記憶體4 GB RAM
  • 顯示卡NVIDIA GeForce 8600 / AMD Radeon HD 2600 以上
  • 硬碟空間5 GB 可用空間
  • 網路寬頻網路連線
建議規格
  • 作業系統Windows 10 / 11（64 位元）
  • 處理器Intel Core i5 第 8 代以上
  • 記憶體8 GB RAM
  • 顯示卡NVIDIA GeForce GTX 1050 / AMD RX 570 以上
  • 硬碟空間10 GB SSD
  • 網路10 Mbps 以上

常見下載問題

下載連結統一發布於官方 LINE 群組的置頂公告中。請先加入群組，再查看置頂訊息即可找到最新版本下載連結。我們不會在其他非官方管道提供下載。
下載完成後，解壓縮壓縮包（如有），執行安裝程式（Setup.exe）並依畫面指示完成安裝。安裝完成後，開啟資料夾中的「登入器（Launcher）」，系統將自動下載最新補丁。補丁更新完畢後即可點擊「進入遊戲」。
部分防毒軟體對遊戲保護程式或登入器可能產生誤判（False Positive）。請確認您的下載來源為官方 LINE 群組，並將遊戲安裝資料夾加入防毒白名單。如有疑慮，歡迎在 LINE 群組詢問 GM 協助確認。
請依序嘗試以下步驟：①確認網路連線正常；②關閉防火牆或防毒軟體後重試；③以「系統管理員」身份執行登入器；④若問題持續，請至官方 LINE 群組回報，GM 將協助您處理。
目前遊戲客戶端僅支援 Windows 作業系統。Mac 使用者可嘗試透過 Parallels Desktop 或 CrossOver 等虛擬化工具執行，但官方不提供此類設定的技術支援。
